🧭 Flujo Interactivo de WhatsApp

Define la creación, participantes y objetivo del grupo por etapa de ventas.

¿Cuándo se crea?

El cliente solicitó cotización o reunión técnica.

Participantes Mínimos

Ejecutivo comercial, Gerente comercial, **Cliente**.

Acción Esperada / Objetivo

Validar requerimientos, aclarar dudas, mantener cercanía y agilidad en la validación de la solución.

¿Cuándo se crea?

Se recibió el PO / se formalizó la orden.

Participantes Mínimos

Ejecutivo comercial, Gerente comercial, **Ingeniero responsable**, Coordinador o Gerente de Ingeniería, **Cliente**.

Acción Esperada / Objetivo

Coordinar entregables técnicos, confirmar planos y fechas de producción. Asegurar el traspaso de información de Venta a Ejecución.

¿Cuándo se crea?

Se definió la fecha de instalación/envío.

Participantes Mínimos

Participantes de Venta Cerrada **+ Operaciones (Logística)**.

Acción Esperada / Objetivo

Confirmar entrega física, preparar la instalación y resolver contingencias de último momento.

¿Cuándo se crea?

El equipo ya fue instalado y probado.

Participantes Mínimos

Cliente, Ejecutivo comercial, Ingeniería.

Acción Esperada / Objetivo

Se avisa que el grupo quedará abierto **20 días para soporte final** y documentación (protocolo de cierre).

¿Cuándo se crea?

El cliente reporta una falla o necesita asistencia técnica o logística.

Participantes Mínimos

Cliente, Ingeniero responsable, Coordinador/Gerente de Ingeniería, **Logística o Compras (según aplique)**.

Acción Esperada / Objetivo

Atender problema puntual, coordinar visita técnica y/o gestión de refacciones.

🚨 Consideración Clave

Aunque WhatsApp sea el canal de comunicación diaria, **todos los acuerdos, decisiones y tareas formales deben ser registrados en el CRM o Slack** para mantener la trazabilidad oficial, tal como se indica en el anexo de Registro de Acuerdos.
