🚥 Ejemplos de Mensajes con Priorización

Guía para clasificar la urgencia de tus solicitudes internas

💬 ¿Cómo debe estructurarse el mensaje?

Elemento ¿Qué incluir?
👤 Responsable Etiquetar al área o persona receptora (ej: `@compras`).
📝 Descripción Clara y breve de la solicitud o contexto.
🕒 Fecha Límite Fecha exacta o urgencia (si aplica).
🔺 Nivel de Prioridad Indicar **Alta**, **Media** o **Baja** para establecer expectativas.
✅ Confirmación Esperar confirmación de lectura (emoji o texto).

Ejemplos por Nivel de Prioridad

🔴 Prioridad ALTA

**Uso:** Temas urgentes con impacto directo en cliente, producción o facturación. Requiere atención **inmediata** (mismo día o próximas 4h).

Solicitud Original:

@compras Urgente 🚨

Cliente Nestlé validó plano y requiere válvula modelo Z123 para instalación del viernes.

Ya tenemos PO y fecha comprometida.

¿Pueden ayudar a gestionarlo hoy mismo? 🙏

Prioridad: ALTA

Respuesta Esperada (Slack):

✅ Recibido, en proceso.

ó

✅ Lo veo ahora mismo.

🟡 Prioridad MEDIA

**Uso:** Requiere atención en los próximos **2 a 3 días**. No es urgente, pero tiene plazo definido.

Solicitud Original:

@ingenieria

El cliente LALA nos pidió ajustar el plano con las nuevas medidas del túnel.

Lo ocupamos antes del jueves para poder enviar cotización.

Prioridad: MEDIA

Respuesta Esperada (Slack):

👀 Recibido, te lo envío antes del miércoles.

🟢 Prioridad BAJA

**Uso:** Tareas sin urgencia inmediata. Pueden resolverse en el transcurso de la semana o según carga de trabajo.

Solicitud Original:

@finanzas

Cuando tengan oportunidad, favor de revisar si quedó aplicada la nota de crédito del cliente Bimbo (julio).

No es urgente.

Prioridad: BAJA

Respuesta Esperada (Slack):

👍 Lo reviso esta semana.

🔴 Prioridad ALTA: Gestión de No Cumplimiento

Si el tiempo de respuesta no se puede cumplir, se debe **comunicar proactivamente** la nueva fecha y escalar, si es necesario.

Solicitud Original:

@ingenieria Urgente 🚨

Cliente Pepsico confirmó instalación para este viernes 13/oct. Necesitamos el plano con las medidas ajustadas para enviarlo hoy mismo a Logística. Prioridad: ALTA. ¿Nos puedes apoyar antes de las 3:00 pm? 🙏

Respuesta Correcta del Área Receptora:

👀 Hola @comercial.pep.

Revisé el caso, pero por carga actual **no podré entregarlo hoy**. El ajuste requiere validación técnica y lo tendría listo el **lunes 16/oct a primera hora**.

Si el cliente no puede mover la fecha, dime y **buscamos con Dirección una solución alternativa.**
